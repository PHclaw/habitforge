> 像锻造铁器一样锻造习惯。每天打卡，积累连击，看着你的习惯等级逐步提升。

## ✨ 功能

- ✅ 每日习惯追踪 — 自定义习惯，每天勾选
- 🔥 连击计数 — 连续天数越长，奖励越高
- ⏰ 提醒功能 — 再也不会忘记打卡
- 📊 数据统计 — 完成率、热力图一目了然
- 🏆 等级系统 — 习惯锻造等级，升级解锁成就
- 📱 响应式设计 — 手机 / 平板 / 桌面

## 🚀 快速开始

```bash
git clone https://github.com/PHclaw/HabitForge.git
cd HabitForge
npm install
npm run dev
```
